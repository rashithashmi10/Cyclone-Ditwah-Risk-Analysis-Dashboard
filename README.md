# 🚨 Cyclone Ditwa Impact & Flood Risk Analysis Dashboard – Sri Lanka 

 My recent Power BI dashboard analyzes the impact of Cyclone Ditwah and assesses population exposure to flood risk across provinces and districts.

🔍 Key Insights:

 • Central Province was identified as the most highly affected region
 
 • Over 270K people are potentially exposed to flood risk
 
 • Western and North Western provinces show high exposure levels
 
 • Flood impact varies significantly across districts, highlighting key risk clusters

🛠 Tools Used:

 • Power BI (Data Modeling, DAX, and Visualization)
 
 • Excel (Data Preparation)

📊 This project focuses on turning data into actionable insights that can support disaster response and planning.
I’m continuously improving my dashboard design skills — feedback is always welcome!

Data Source - United Nations Satellite Centre (UNOSAT)
